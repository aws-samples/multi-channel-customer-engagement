# Prerequisites

For this walkthrough, you should have the following prerequisites:

    An AWS account
    Access to the following AWS services:
        Amazon Connect
        Amazon Lex
        Amazon Comprehend
        Amazon Transcribe
        Amazon S3
        Amazon DynamoDB
        Amazon API Gateway
        Amazon Cognito
        AWS Lambda
        AWS Step Functions
        Amazon CloudFront
        Amazon PinPoint
        Amazon Kinesis
        Amazon SNS
    Basic knowledge of Python and Javascript
    An Amazon Alexa Developer account
